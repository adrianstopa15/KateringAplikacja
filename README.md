# Aplikacja do obsługi kateringów

Aplikacja webowa do obsługi firm kateringowych, umożliwiająca klientom zamawianie diet zgodnie z preferencjami oraz firmom kateringowym zarządzanie ofertą posiłków i diet po akceptacji administratora.

# Główne funkcjonalności

### Klient:
- rejestracja i logowanie
- wybór preferencji żywieniowych
- składanie zamówień na diety
- podgląd historii zamówień

### Firma kateringowa:
- tworzenie konta firmowego
- dodawanie diet i posiłków
- edycja własnej oferty

### Administrator:
- akceptacja diet i posiłków firm
- zarządzanie użytkownikami
- nadzór nad systemem

# Stack technologiczny

- React

### Backend:

- Spring Boot
- Spring Security

### Baza danych:
